# Code of Conduct

CNMSQL follows the [CNCF Community Code of Conduct](https://github.com/cncf/foundation/blob/main/code-of-conduct.md), which is itself adapted from the [Contributor Covenant](https://www.contributor-covenant.org/version/2/0/code_of_conduct/).

Everyone who takes part in the project is expected to follow it. That covers issues, pull requests, discussions, and any other space connected to the project. The point is to keep this a place where people can work together without harassment, whoever they are.

## Reporting

If you see or experience behavior that breaks the code of conduct, report it to the maintainers at conduct@cnmsql.co. Reports are handled in confidence. The maintainers will look into every report and respond as the situation warrants, up to and including removing someone from the project's spaces.

The full text of the policy, including the standards we hold each other to and the enforcement process, is in the [CNCF Community Code of Conduct](https://github.com/cncf/foundation/blob/main/code-of-conduct.md).
